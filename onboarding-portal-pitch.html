Internal Project Proposal

Customer Onboarding Portal

Reducing friction in site onboarding and credential collection

The Problem

Onboarding is our biggest time sink

When a new customer wants to connect their solar sites to Metris, they currently face a fragmented, confusing process that creates friction for them and manual work for us.

Current Process

  • Basic form asking "which platforms do you use?"
  • No guidance on how to get API keys
  • Credentials shared via email (insecure)
  • Multiple back-and-forth emails
  • Site details collected separately
  • 5-7 day turnaround

With This Portal

  • Step-by-step guidance for each platform
  • Video walkthroughs showing exactly how
  • Secure credential submission
  • Self-service, no emails needed
  • Site + credentials in one unified flow
  • 1-2 day turnaround
The Solution

A unified, self-service onboarding experience

I built a customer-facing portal that combines site onboarding with credential collection. Customers can add their site details, select their monitoring platforms, and get guided through exactly how to obtain and share API credentials — all in one flow.

The key insight: customers don't know how to get API keys. They need hand-holding through each platform's portal. This tool does that.
